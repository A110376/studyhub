📘 StudyHub – PPSC Preparation Platform

StudyHub is a modern web application designed to help students prepare for PPSC (Punjab Public Service Commission) exams. It provides mock tests, MCQs practice, and progress tracking to ensure effective and structured learning.

🚀 Features
🔐 User Authentication
Sign up & Login system
📝 Mock Tests
Simulated exam environment
Helps in real-time preparation
📚 MCQs Practice
Topic-based learning
Improve concepts through practice
📊 Progress Tracking
Visual progress charts
Track performance over time
⚡ Responsive UI
Clean and user-friendly design using Tailwind CSS
🛠️ Tech Stack

Frontend:

React.js
Tailwind CSS

Backend:

Python (Django / Flask – customize as per your project)

Database:

PostgreSQL
